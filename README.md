# Flutter Music Player (YouTube Integration)

แอปพลิเคชันเล่นเพลงที่ดึงข้อมูลจาก YouTube Data API และเล่นวิดีโอ/เสียงผ่าน YouTube Player

## คุณสมบัติ (Features)
- ดึงรายการเพลงใหม่ล่าสุดจาก YouTube โดยอัตโนมัติ
- ระบบจัดการสถานะด้วย **BLoC Pattern**
- หน้าเล่นเพลงแบบละเอียดที่ใช้ **SliverAppBar** เพื่อการเลื่อนที่ลื่นไหล (Collapsing Header)
- ปุ่มควบคุมพื้นฐาน: Play/Pause, Skip Next และระบบวนเพลงอัตโนมัติ
- แท็บแสดงคิวเพลง (UP NEXT) และส่วนเตรียมรองรับเนื้อเพลง (LYRICS)

## เทคโนโลยีที่ใช้ (Tech Stack)
- **Flutter** (Framework)
- **flutter_bloc**: สำหรับ State Management
- **youtube_player_flutter**: สำหรับเล่นเนื้อหาจาก YouTube
- **http**: สำหรับเชื่อมต่อกับ YouTube Data API
